# reasoning_image_generation

多模态大型语言模型（MLLM）的最新进展在许多推理任务中取得了显著进展，但在抽象视觉推理任务中仍然失败。为此我们提出纯规则（non-ML）数据合成管线，分为两类任务：

单图题（类识别）：以几何构件为基点，生成两类变式——（1）单形状内部绘制线条或局部遮盖产生变形；（2）多形状的嵌套 / 邻接 / 相交组合以生成同类判定难例。
多图序列题（序列推断）：采用网格 / 扇形 / 随机布局生成图像序列，随机采样原图后通过平移、旋转、去同存异、去异存同、单一遍历等操作生成有规律的序列与候选项，任务为预测下一个项。

通过批量程序化渲染并过滤语义重复样本，输出 ≈1,000 条高质量训练样本；在 `qwen2.5-vl-7B` 上进行 SFT 实验，在 MathVision 上获得约 **+5pp** 的准确率提升，验证了规则合成样本依旧能对先进模型的抽象视觉推理能力的有效补强。
